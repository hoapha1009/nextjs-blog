# Next Blog

A Blog build with NextJS, Markdown and stored with MongoDB

# Technology

NextJS, MongoDB, Markdown, Module CSS, HTML5, CSS,...

# Link demo

https://nextjs-blog-sigma-roan.vercel.app/

# Using

<ol>
<li>
    Clone this repo to your computer
</li>
<li>
    Run `npm install` or `yarn` in your terminal
</li>
<li>
    Run `yarn start` or `npm start`
</li>
